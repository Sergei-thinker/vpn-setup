# Last updated: 2026-04-08
# ============================================
#
# HOW TO APPLY IN HIDDIFY:
#
# Method 1: Via Hiddify Settings UI
#   1. Open Hiddify -> Settings -> Config Options -> Routing
#   2. Set "Remote DNS" to: https://1.1.1.1/dns-query
#   3. Set "Direct DNS" to: https://77.88.8.8/dns-query (Yandex DNS for Russian domains)
#   4. Under "Routing Mode" select "Custom"
#   5. In "Direct domains / keywords" paste the domain list below
#   6. In "Direct IPs" enter: geoip:ru, geoip:private
#   7. Leave "Blocked domains" empty or add ad domains
#   8. Save and reconnect
#
# Method 2: Via Subscription Link
#   If your 3X-UI panel generates Hiddify subscription links, server-side
#   routing (xray-server-routing.json) will handle split routing automatically.
#
# Method 3: Via Custom Config
#   1. Open Hiddify -> Settings -> Config Options -> Advanced
#   2. Enable "Custom Config"
#   3. Paste the Xray routing JSON from xray-server-routing.json
#
